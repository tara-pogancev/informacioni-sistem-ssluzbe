Uputstvo za korišćenje aplikacije: Informacioni Sistem Studentske Službe

1. Prečice na tastaturi

	New: Ctrl + N
	Close: Ctrl + C
	Edit: Ctrl + E
	Delete: Ctrl + D
	Help: Alt + F1
	About: Alt + A
	Search: Ctrl + S

	File: Alt + N
